# Function-Level API Documentation

This folder contains detailed documentation for each Edge Function/API endpoint.

## Template

Each function document should include:
- Overview and endpoint info
- Flow diagram (ASCII)
- Request/Response schemas with TypeScript interfaces
- Database tables read/written with SQL queries
- Storage buckets used
- Error codes
- Cost breakdown (if applicable)
- Deployment instructions

See the Function Doc Template in CLAUDE_TEMPLATE.md for the full structure.
